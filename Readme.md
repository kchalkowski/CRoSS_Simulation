# Caribou-Road Systems Science (CRoSS) Simulation    
The purpose of this repository is to develop a predictive simulation on caribou movements in response to roads.     

# Current status
* Developed very simple framework based on movement between two defined range polygons

# Next steps
* Allow for user input in targets script to change range polygon parameters
* Convert gamma-distributed step-length random walk to correlated velocity model movement (using example tau and speed parameters)
* Create algorithm to change movement process dependent on road intersection
